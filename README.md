# Boilerplate Express + TypeScript

### Prerequisites

- Node@v10.x.x

### Install

```bash
$ git clone https://github.com/crisantizan/express-ts-boilerplate
$ cd express-ts-boilerplate
$ yarn
```

### Scripts

- **start**: run in production
- **serve**: run in development
